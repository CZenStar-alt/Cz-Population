## Installation
- Download from Code.
- Extract and Drop at Resource Folder.
- Update Below in your Server CFG file :
ensure Cz-Population
- Adjust Population as per your choice.
1= 100%, 0.5= 50% and so on.

- Done
- Enjoy.
